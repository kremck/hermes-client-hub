# hermes-client-hub
A client &amp; lead tracking web app, created for the News-Register sales team.

**Overview**

This is a browser based web app that allows the user to keep track of clients, leads, and ads being ran.

**Instructions**

Since this isn't a high-stakes tool, the "login" system is purely honor-based, meaning that you just select your name and profession without a password being required.

Users are stored via the shared drives that already exist.  Users can be added and removed in the Users tab.

_For Sales People_
You can create new reminders under Quick Add.  You will be able to see the reminders and clients that you added, and others are scoped only to the user that made them.

_For Graphics People_
Graphics has access to this tool for the sole purpose of including the final proofs that can be provided to clients.
